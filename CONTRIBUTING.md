# Contributing to the Roadmap
We love contributions! To keep the quality high, please ensure:
1. The tool is actually Open Source (not just "Free tier").
2. The GitHub repository is active (commits within the last 6 months).
3. You include the star badge using the format in the README.

To add a tool: 
- Fork the repo.
- Add the tool to the correct category in alphabetical order.
- Open a Pull Request!
